# Session record: 2026-06-13

## Decision: NDE semantic design Revision 2 ratified (operator)

The operator ratified the NDE semantic design, Revision 2
(NDE-SEMANTIC-DESIGN.md, in the NDE repository). Revision 2 had been
issued 2026-06-12 as DRAFT for re-ratification after Revision 1 was
ratified and reopened the same day; see docs/sessions/2026-06-12.md
Addendum 2 for the six corrections (A through F). This ratification
is the pivot to focus on a native desktop environment built to fit
Awase.

What this ratifies, per the document's section 9:

- Decisions 1, 3, and 4 stand as ratified (layout intent with
  resolver-computed geometry; bare fenced canvas at C2 v1 with
  temporal bindings deferred to M5; tree-only non-canvas UI for
  native applications).
- Decision 2 is re-ratified in its Revision 2 form: the tree and
  transaction encoding is a versioned wire format and reusable
  library in Awase shared/, transport terminating at NDE, vocabulary
  in NDE, with LT-1 remaining pure presentation mechanism. Revision
  1's disposition (the tree carried on LT-1 itself) is withdrawn as
  a design error.
- Decisions 5 (the v1 core vocabulary, role by role) and 6
  (tree-read authority gate) remain open for the C2 ADR series.

Recorded in the NDE repository this session: the
NDE-SEMANTIC-DESIGN.md status block flipped from "Revision 2, DRAFT
for re-ratification" to "Revision 2, Ratified 2026-06-13 (operator)".

## Consequences executed in this tree

- LT-1 (BACKLOG.md) gains a scope note: LT-1 is presentation
  mechanism only (transform, opacity, clip, z-order); the semantic
  tree is unbundled and lives as a separate shared/ wire format and
  library terminating at NDE, never fused with LT-1. This closes the
  LT-1 review owed in docs/sessions/2026-06-12.md.
- SM-5 (BACKLOG.md) closed. The quick-exit guard false-positive on
  clean logout was verified fixed in pgsd-sessiond/src/main.zig: the
  session-loop guard gates on child_exit != 0, so a clean code-0
  logout is silent at any duration. The landed code matches the
  entry's Fix paragraph; the inline comment at the guard names SM-5.
